# داشبورد KPI — لیگ ۲۴ عیار اکسیراز

داشبورد آنالیتیکس کمپین پیش‌بینی **جام جهانی ۲۰۲۶** اکسیراز («لیگ ۲۴ عیار»).
پروژه‌ی **مستقل، استاتیک و بدون build** — فقط HTML/CSS/JS. هیچ نصب یا سروری لازم نیست.

> شعار: «جایی که پیش‌بینی تو، طلا میشه!» · تگ‌لاین: «با پیش‌بینی‌های درست، شمش ۲۴ عیار ببر»

---

## اجرا

فایل `index.html` را مستقیم در مرورگر باز کن. تمام.
(برای رندر نمودارها به اینترنت نیاز است چون Chart.js و فونت از CDN می‌آیند — برای حالت آفلاین به بخش «آفلاین» پایین نگاه کن.)

---

## ساختار فایل‌ها

```
24CAMPAIGN/
├── index.html          اسکلت صفحه (دست نزن مگر بخش جدید بخواهی)
├── README.md           همین فایل
├── css/
│   └── dashboard.css   تم طلایی/تیره، RTL، ریسپانسیو
└── js/
    ├── data.js         ⬅️ تنها فایلی که برای به‌روزرسانی دستی پر می‌کنی
    └── dashboard.js    منطق رندر و نمودارها (دست نزن)
```

**قانون طلایی:** برای به‌روزرسانی اعداد فقط `js/data.js` را عوض کن. بقیه‌ی محاسبات
(CPA، نرخ تبدیل، درصدها، مجموع‌ها) خودکار انجام می‌شود.

---

## چطور داده وارد کنم؟ (`js/data.js`)

داخل آبجکت `CAMPAIGN_DATA` فقط عددِ مقابل هر کلید را عوض کن. نکته‌ها:

- مبالغ به **تومان**.
- اعداد را **لاتین** بنویس (`1250` نه `۱۲۵۰`) — نمایش فارسی خودکار است.
- برای افزودن یک روز جدید، یک آبجکت به آرایه‌ی `daily` اضافه کن.

### دیکشنری داده

| کلید | چیست | توضیح |
|------|------|-------|
| `meta` | اطلاعات کلی کمپین | روز جاری، بازی‌های برگزارشده، ددلاین بعدی، تاریخ به‌روزرسانی |
| `kpis` | شاخص‌های اصلی | ثبت‌نام، کاربر فعال، کل پیش‌بینی، بازدید، هزینه |
| `daily[]` | روند روزانه | هر ردیف = یک روز؛ برای نمودار خطی |
| `funnel` | قیف اکتساب | بازدید → ثبت‌نام → اولین پیش‌بینی → کاربر بازگشتی |
| `channels[]` | عملکرد کانال‌ها | بر اساس `utm_source`؛ هزینه/کلیک/ثبت‌نام |
| `predictionTypes[]` | تفکیک تعامل | پیش‌بینی به تفکیک نوع (دونات) |
| `retention` | نگهداشت | دعوت، هت‌تریک، اکسیر، حضور متوالی |
| `users` | کاربر جدید/قدیمی | منبع رشد پایگاه کاربری |
| `iran` | پیش‌بینی‌های ویژه ایران | صعود، اولین گلزن، بهترین نتیجه |

> اگر مقداری را صفر بگذاری، آن بخش حالت «خالی» نشان می‌دهد (خطا نمی‌دهد).

## وضعیت داده‌ها (آخرین به‌روزرسانی)

| متریک | مقدار | بازه | یادداشت |
|-------|-------|------|---------|
| رویداد ثبت‌نام | **۱۴٬۵۲۳** | ۱۴۰۵/۰۳/۱۹ تا ۱۴۰۵/۰۴/۰۱ | ⚠️ هنوز مشخص نیست فقط «ورودی کمپین جام جهانی» است یا کل ثبت‌نام‌ها |

**باز/در انتظار داده:** پیش‌بینی‌ها و تعامل، بازدید صفحه (برای نرخ تبدیل)، داده‌ی روزانه،
کلیک/ثبت‌نام هر کانال، نگهداشت، کاربر جدید/قدیمی، پیش‌بینی‌های ویژه ایران.

---

## بخش‌های داشبورد

1. **هدر کمپین** — نام/شعار، روز X از ۳۹، بازی X از ۱۰۴، ددلاین بعدی، نوار پیشرفت
2. **کارت‌های KPI** — ثبت‌نام، DAU، کل پیش‌بینی، نرخ تبدیل، هزینه کل، CPA
3. **روند روزانه** — نمودار خطی ثبت‌نام/پیش‌بینی/کاربر فعال
4. **قیف اکتساب** — از بازدید تا کاربر بازگشتی
5. **عملکرد کانال‌ها** — دونات سهم + جدول کامل با CPA و نرخ تبدیل
6. **تعامل** — انواع پیش‌بینی + متریک‌های نگهداشت
7. **کاربران جدید/قدیمی** + بلوک ویژه‌ی **تیم ملی ایران**

---

## فاز بعد (اتوماتیک‌سازی)

الان داده دستی در `data.js` وارد می‌شود. برای اتصال زنده به شیت گوگل کافی است
`data.js` با یک `fetch` از خروجی Publish-to-web (CSV) یا Sheets API جایگزین شود؛
ساختار `CAMPAIGN_DATA` همین می‌ماند تا `dashboard.js` بدون تغییر کار کند.

## آفلاین (اختیاری)

برای کارکرد بدون اینترنت، فایل `chart.umd.min.js` و فونت Vazirmatn را لوکال در یک
پوشه‌ی `vendor/` کپی کن و مسیرهای CDN را در `index.html` به آن تغییر بده.
